# Examples

This directory is reserved for small, shareable examples that illustrate the pipeline without copying large corpora or generated datasets.

Possible future contents:

- A tiny synthetic boundary table showing expected column names.
- One anonymized or license-compatible TextGrid/WAV example.
- A minimal command sequence for reproducing a single case visualization.

Do not place full JVS, LibriTTS, MFA output directories, or large generated CSVs here unless licensing and repository size constraints have been addressed.
